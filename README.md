# Система управления рисками организации (СУР)

Инструмент риск-менеджера для ведения единого реестра рисков, планирования мероприятий по их обработке и визуализации на тепловой карте — реализован в виде книги Excel (`.xlsx`) с формулами и условным форматированием, без надстроек и макросов.

Методологическая база: **ISO 31000:2018** и **COSO ERM 2017**.

## Состав

- **`Система_управления_рисками.xlsx`** — рабочая книга (5 листов: Инструкция, Справочники, Реестр рисков, План обработки рисков, Матрица рисков).
- **`prd.md`** — техническое задание (Product Requirements Document), по которому собрана книга.

## Быстрый старт

1. Откройте `Система_управления_рисками.xlsx` в Excel или LibreOffice Calc.
2. Начните с листа **«Инструкция»** — там описан состав книги и порядок работы (5 шагов).
3. При необходимости адаптируйте шкалы на листе **«Справочники»** под специфику организации.
4. Ведите риски на листе **«Реестр рисков»**, мероприятия — на листе **«План обработки рисков»**.
5. Используйте лист **«Матрица рисков»** для приоритизации и отчётности руководству.

Подробные функциональные и нефункциональные требования — в [`prd.md`](prd.md).
